# Outline Structures

Skeleton structures for each output format. Use as starting points, adapt based on content.

---

## Long-Form (600-800 words)

### Structure A: Transformative Arc
```
HOOK (50-75 words)
- Opening observation/provocation
- Recognition moment

PROBLEM (100-125 words)
- Surface the pain
- Hidden costs
- Emotional weight

ROOT CAUSE (125-150 words)
- The mechanism behind the symptoms
- Why this happens
- "Here's what's actually going on"

REFRAME (125-150 words)
- New mental model
- The perspective shift
- Why this changes everything

PATH FORWARD (150-175 words)
- Practical framework(s)
- Principles, not prescriptions
- What to do first

CLOSE (50-75 words)
- Transformation vision
- Strategic question
- Return to opening with new weight
```

### Structure B: Strategic Oracle Arc
```
SIGNALS (100-125 words)
- 2-3 disconnected observations
- Factual, current, verifiable
- Hint at connection

PATTERN SYNTHESIS (200-225 words)
- What these signals reveal
- Historical pattern connection
- Evidence building
- "Which means..."

PROBABLE FUTURES (150-175 words)
- Where this leads
- Behavioral/structural changes
- Probabilistic language

STRATEGIC STAKES (100-125 words)
- Competitive advantage of seeing this
- The choice facing reader
- Callback to opening signals
```

### Structure C: Decoder Arc
```
HOOK (75-100 words)
- Present the concept/framework
- Why understanding mechanics matters
- Promise revelation

MECHANICS (350-400 words)
- Break down each component
- Show the logic/math/physics
- Why it works this way
- Where it breaks down
- Practical implications within

STRATEGIC UNDERSTANDING (125-150 words)
- Synthesis of the complete picture
- How to use this understanding
- Competitive positioning
```

## Medium (LinkedIn ~350 words)

### Structure: Single Insight Focus
```
HOOK (30-50 words)
- Provocative opening
- Pattern interrupt
- Before "see more" button: tension established

INSIGHT (150-175 words)
- The core idea
- One supporting proof point
- "Here's what this means..."

IMPLICATION (75-100 words)
- So what?
- What changes with this understanding
- Reader-facing consequence

CLOSE (30-50 words)
- Question or call to conversation
- Return to hook
- Energy to engage
```

### Alternative: Story → Lesson
```
HOOK (30-40 words)
- Story opener or observation

STORY/EXAMPLE (125-150 words)
- Specific, concrete narrative
- Details that create recognition

INSIGHT (100-125 words)
- What this reveals
- The principle underneath

CLOSE (40-50 words)
- Application for reader
- Engaging question
```

## Short-Form (Chili/X - Character Limited)

### Structure: Tension → Resolution
```
LINE 1: The tension/hook
- Surprising claim
- Contrarian take
- Pattern observation

LINE 2-3: The insight
- What this means
- Why it matters

LINE 4: The landing
- Implication
- Question
- Sharp close
```

### Patterns That Work:

**The Inversion:**
"Everyone thinks X. But actually Y. Because Z."

**The Observation:**
"[Specific thing] is [surprising insight]. [Why this matters]."

**The Contrast:**
"[Old way] doesn't work anymore. [New reality] because [reason]."

**The Question:**
"What if [assumption] is wrong? [Implication]."

## Outline Quality Checklist

Before presenting outline for approval:

**Structure:**
- [ ] Clear progression from opening to close
- [ ] Each section has distinct purpose
- [ ] Proportions appropriate for format
- [ ] Transitions feel natural, not forced

**Content:**
- [ ] Single clear thesis/insight
- [ ] Specific examples or evidence identified
- [ ] Personal angle or perspective clear
- [ ] Action/implication defined

**Voice:**
- [ ] Opening creates energy
- [ ] Close lands with weight
- [ ] Emotional journey considered
- [ ] Framework serves content (not reverse)

---

## Adapting Structures

**When to deviate:**
- Content doesn't fit the arc → choose different structure
- One section needs more space → adjust proportions
- Insight is simple → shorter piece, not padded long one
- Multiple strong insights → separate pieces, not one bloated one

**Warning signs:**
- Forcing content into structure
- Padding to hit word count
- Repetition to fill sections
- Transitions that feel mechanical
